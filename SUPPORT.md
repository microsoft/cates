# Support

## How to file issues and get help

This project uses [GitHub Issues](https://github.com/microsoft/cates/issues) to track bugs and feature
requests. Please search the existing issues before filing new ones to avoid duplicates. For new
issues, file your bug or feature request as a new Issue.

For help and questions about using this project, please open a [GitHub Discussion](https://github.com/microsoft/cates/discussions)
or file an issue with the `question` label.

## Microsoft Support Policy

Support for **cates-analyzer** is limited to the resources listed above. This project is provided
as-is under the [MIT License](LICENSE) and is not covered by Microsoft Premier or Standard Support
agreements.
